This is the client library meant to be imported by users apiserver-network-proxy.
The primary user of this library is the kubernetes-apiserver from https://github.com/kubernetes/kubernetes. 
The dependencies for this library should be kept in line with the old supported version of Kubernetes.
The oldest supported version of kubernetes can be found at https://kubernetes.io/releases/

Please do not make changes in apiserver-network-proxy/konnectivity-client and anywhere else in apiserver-network-proxy.
